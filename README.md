# 不同平台的驱动适配模版
本项目旨在提供不同平台的驱动适配教程，目前驱动包括屏幕和sensor，涵盖了海思和瑞芯微等多个平台，教程不仅提供了驱动适配的指导，还包括基本的配置指南，以帮助用户在不同平台上成功配置和使用这些驱动。

## 支持的平台包括

```
VariousPlatformDriverAdaptation/

```
